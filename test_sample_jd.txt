ROLE OVERVIEW
====================================

We are seeking a Senior Backend Engineer to join our growing platform team. You will be responsible for designing, building, and maintaining scalable backend systems that process millions of data points daily. You'll work with cutting-edge technologies and collaborate with a talented team of engineers.

====================================
KEY RESPONSIBILITIES
====================================

- Design and develop microservices using Python and FastAPI
- Build and optimize complex SQL queries for large-scale databases (PostgreSQL, MySQL)
- Implement caching strategies using Redis to improve system performance
- Set up and maintain Docker containerization for applications
- Manage AWS infrastructure including EC2, RDS, S3, and Lambda
- Implement CI/CD pipelines using GitHub Actions or similar tools
- Write clean, testable code with comprehensive unit tests (Pytest)
- Collaborate with frontend engineers and data scientists
- Participate in code reviews and mentor junior developers
- Troubleshoot and debug production issues
