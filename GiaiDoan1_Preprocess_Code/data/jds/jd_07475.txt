Job Description The Cloud Security Engineer/Architect is to develop and maintain the high- level technical design plans for the Office 365 cloud security architecture. This individual will provide technical consulting and recommendations across the Office 365 domain and related technologies. The Cloud Security Architect employs deep technical knowledge to design, build, and operate and enhance systems. The Cloud Security Engineer/Architect is a technical consultant within the Cloud Security architecture team. As a Cloud Security Engineer/Architect, you will focus on security for Microsoft Office 365, and the associated integrations with the Azure platforms to provide: Security assessments of client Office 365 implementations or planned implementations Insight and input into the Cloud Security Architecture team to build new service offerings and security findings for Office 365 Security assessments of related SaaS products and services Perform SaaS security assessments using domain knowledge and automated tools (CASB etc)
