The Enterprise Cloud Native Infrastructure Architect position in the HPE Global IT Enterprise Architecture organization is being created to lead the strategy, design, and migration to HPE’s IT new hybrid cloud hosting built on HPE’s best in class solutions. New application development will focus on containerized, cloud native micro services and applications. The object is to implement a hybrid cloud integration platform strategy, with self-service delivery in support of HPE enterprise digital business needs. In this role, the architect will work closely with the Infrastructure operations team to put in place the required hosting platforms and automation to enable self-service capabilities. In addition, the architect will work closely with application teams providing best practices and infrastructure consulting for new and existing application migrating to the new cloud native hosting platforms. You will help develop our cloud application development and migration methodology. You will also implement standards and solutions for developing and deploying PaaS workloads based on DevOps, Continuous Development, Continuous Deployment, and Continuous Integration.
