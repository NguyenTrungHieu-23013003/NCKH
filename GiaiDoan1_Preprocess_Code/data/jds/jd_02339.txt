Overview The Database Administrator/SQL Developer is responsible for creation and maintenance of the Database Development Life Cycle. This position requires experience in software engineering and expertise with Postgres. This position requires communication with other departments and the ability to drive projects and issues to completion. The Database Administrator must develop mastery of all aspects of the Database Development Life Cycle.. Responsibilities Develop detailed and thorough knowledge of database schema, database operations and database communication layers; Design large scale changes to database technology as needed Develop a thorough and detailed understanding of the relational database platforms, with a focus on performance analysis, query optimization operations, appropriate SQL syntax and database internals Perform independent database performance analysis with a focus on SQL query efficiency and schema design; apply tools and utilities to identify integrity or performance issues Create white papers from research to support database initiatives; create internal documentation for educating and training employees Work with other development teams to review database queries and advise use of best practices using common data layer Maintain Database Development Lifecycle Review database schema changes and generated queries as needed; assist and mentor developers with schema and query reviews Work with team to enforce database standards; aid in design and research of new database standards as needed Qualifications 5+ years of experience analyzing queries and making table recommendations Clear, fundamental understanding of relational databases, best practices and industry-recognized concepts pertaining to Postgres, Relational Database Management System platforms and SQL. Knowledge of Linux and Windows operating systems Demonstrates experience in OOP concepts; Demonstrates experience in either C++ or C languages; Familiarity with supported database communication frameworks Curate and edit internal records; Regular internal speaking engagements to communicate critical database concepts, best practices and new features Able to articulate database solutions to developers and get consensus on the proper solution Proven ability to stay current on latest database trends and topic Self-motivated with the ability to manage projects to completion with minimal oversight Ability to own solutions and ensure that they continue to work with changing customer environments
