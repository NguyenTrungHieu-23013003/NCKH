Overview GovernmentCIO is looking for a System Administrator to join our team. The role is part of the Genomic Information System for Integrated Science (GENSIS) project at the VA Medical Center in Boston, MA. The System Administrator serves as a Subject Matter Expert (SME) that provides data center, infrastructure, network, and communications technical leadership, guidance and recommendations. Provides input to the project architects required to make critical decisions that define a specific direction for a system in terms of implementation, operations, and maintenance. Responsibilities The Software Developer will: Lead the Infrastructure Change Control Board meetings (CCB) Develop and maintain Environment design document Manage migration to new Dev/SQA environments Make recommendations on upgrades OS, database, Java, and appliance firmware versions Manage Certificate Tracking and Implementation SOPs for each environment Review and/or create scripts to assist in environment configuration management Ability to make recommendations on database designs, database replications plans, and database cleanup activities Knowledge of a system’s Authorization to Operate (ATO) process. Ability to make recommendations on new server specifications when provided by Dev Teams Enforce operational, release, and configuration processes and procedures Create and maintain software roadmaps and interdependencies of infrastructure layer software with application layer software Perform COTS product tools analysis and recommendations for acquisition consideration by the Data Center Managers Lead the review of all server planning sheet submissions, proactively make recommendations for change to as-built configurations Qualifications Required: Bachelors Degree in computer science, electronics engineering or other engineering or technical discipline is required At least 5 year’s experience managing multiple enterprise technical environments. Experience with Wildfly, Apache web server, MSSQL Familiarity with Apache SOLR, Apache Jena; is a plus Enterprise understanding of data center services to include: hardware and software, hosting services, and network communications Leadership skills, initiative, and good judgment. Good communications, analytical, and writing skills. Ability to work independently within a team-oriented environment Competency in MS Word, MS Visio, MS Excel, and MS SharePoint
