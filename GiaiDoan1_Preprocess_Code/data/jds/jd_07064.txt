The Hyperion System Administrator position supports and maintains Oracle Hyperion EPM systems, day-to-day activities includes the infrastructure and application administration, configuration, patching, backup, recovery and maintenance of Hyperion systems and products DUTIES AND RESPONSIBILITIES. Expert in troubleshooting performance related issues. Debug Essbase, Planning, HFM, FDMEE and FCM/ARM issues. Manage the backups for Essbase, Planning, EPMA, HFM, FDMEE and FCM/ARM using LCM and Hot/Cold backup methods for Essbase. User and group administration for all the applications. Updating FDMEE mappings and monitoring the batch jobs. Migrating the application artifacts across the instances using LCM. Troubleshoot and resolve the user issues using the ticket monitoring system. Resolve Smart view issues. System monitoring for all the environments and month end close support. In depth knowledge in load balancing methods and server clusters. Document the configuration and application changes. Provide required documentation for SOX/Internal audit teams. Qualifications MINIMUM QUALIFICATIONS / SKILLS Minimum 7 year of experience as Hyperion admin - install, maintain, configure, patch, cluster setup of EPM environments Experience in Linux shell and windows batch scripting. Strong knowledge in EPM 11.1.2.4 products. Must have in-depth experience with installation and optimization of the Oracle Hyperion EPM Suite across multiple environments with cluster configuration. PREFERRED QUALIFICATIONS Bachelors / Master degree in computer science or related field PHYSICAL REQUIREMENTS / WORKING CONDITIONS Office environment While performing the duties of this job, the employee is frequently required to sit. The employee is occasionally required to stand, walk and use hands. The employee may occasionally lift and/or move up to 10 pounds. Specific vision abilities required by this job includes close vision and distance vision.
