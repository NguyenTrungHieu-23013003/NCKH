The Windows Systems Administrator is responsible for the upkeep, configuration, and reliable operation for Windows computer systems. The Systems Administrator seeks to ensure that the uptime, performance, resources, and security of the computers they manage meet the needs of the users. To meet these needs, the System Administrator will install, or upgrade computer components and software; provide routine automation; maintain security policies; troubleshoot; educate, and/or offer technical support for projects. Job Responsibilities: Installs new software releases and system upgrades, evaluates and installs patches, and resolves software related problems Performs system backups and recovery Maintains data files and monitors system configuration to ensure data integrity Supporting technologies including: Operating systems Servers Networks System security Hardware Other third-party software applications Troubleshoots system issues and resolves or escalates Creates and updates system documentation, including configurations and backup procedures Job Qualifications: Must have 5-7+ years’ of Windows Server Administrator / Windows System Administrator Experience Must have strong experience with Windows Server 2012/2016 Must have strong Microsoft Windows System Administrator experience Must have strong Active Directory experience Must have experience with VMware Must have a basic understanding of EMC storage arrays Job Qualifications Pluses: Implementing VMware VxRail – plus to have Implementing Office 365 (O365) – plus to have Experience with MobileIron would be a plus Experience with Symantec Enterprise Vault (Symantec eVault) would be a plus Some Linux would be a plus as well Plus to have the Microsoft Certified Systems Administrator (MCSA) Plus to have the Microsoft Certified Systems Engineer (MCSE) Plus for Bachelors of Science in Information Technology and/or Networking Job Title Also Known as (AKA): System Administrator Systems Administrator Windows Systems Administrator – Windows Windows Server Administrator Windows System Administrator Windows – System Administrator Microsoft Windows System Administrator NOTE: Any offer of employment is contingent on passing a drug test and a background screen prior to starting.
