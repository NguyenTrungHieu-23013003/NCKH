The Data Center Operations Engineer position is responsible for supporting Data Center environments in multiple locations. This includes the installation and maintenance of server and networking hardware, cabling, inventory and incident management. This position requires someone that can work independently on assigned tasks as well as interact with multiple internal IT groups and hardware vendors. Core experience requirements: 3 to 5 years of experience working in a data center environment. Familiar with data center components such as network racks, server enclosures, PDUs, copper & fiber cabling. Excellent verbal and written communication skills. Additional preferred experience: Familiarity with HP server & blade chassis hardware capabilities. Good understanding of Power/Heating/Cooling for server racking purposes. Experience with Incident Handling and responding to service requests. Experience racking and cabling blade chassis and servers including hardware maintenance. Responsibilities: Maintain and install hardware in multiple data centers. Detailed oriented & complete all activities in accordance to IT guidelines. Troubleshoot copper and fiber cabling including initial installation testing as well as connectivity issues. Perform hardware upgrades and initiate hardware vendor support as necessary. Provide on-site support and monitor the Data Center environment. Interact with various IT groups and assist in troubleshooting incidents Manage service requests and incidents via ticketing system. Maintain hardware Inventory database of all Data Center hardware. Utilize reporting tools to build new reports as required. Must be able to travel between data centers including overnight stays. Able to lift/move 25 to 50 lbs. daily
