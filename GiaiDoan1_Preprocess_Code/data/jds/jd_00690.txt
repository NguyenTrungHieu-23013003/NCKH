PRIMARY FUNCTION : The Data Warehouse (Integration) Analyst will plan, analyze and verify project solutions to integrate/transform data and information from different departmental systems and hospital networks into a centralized repository Warehouse. The Data Integration Analyst is responsible for taking customer requirements and converting them into data specifications for data mapping and transformation. ESSENTIAL FUNCTIONS: Analyzes requirements and studies system processes and data flows; creates detail specifications to map, translation and standardizes data for data transformation development. Documents data flow processes and frameworks. Executes scripts for data analysis, testing, ranging from unit to integration testing. Creates and maintains supporting documentation, such as data flows, entity relationships, business and process flow. Works to ensure accurate data mapping and data transformation between source systems, data warehouses, and data marts. Assists in data validation and data testing on projects involved; assist in production data check-out and sign-off as needed during data refresh cycles as needed. Interacts with co-workers, visitors, and other staff consistent with the values of Jefferson.
