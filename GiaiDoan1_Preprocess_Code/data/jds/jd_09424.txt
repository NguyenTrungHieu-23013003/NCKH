The Information Security Analyst (Identity & Access) will perform access control and change deployment tasks in IT systems following standard operating procedures and work instructions. This individual will ensure these activities are conducted in compliance with the Post Consumer Brands IT Internal Control Framework and that required records are maintained. The Internal Control Analyst will also work closely with Business Partners and IT Staff to meet service level expectations and support effective and efficient IT processes. Maintain audit compliance for all code & configuration changes o Advise leadership of and remedy any control gaps detected in course of activities o Collaborate with appropriate internal staff and audit staff on remediation activities o Provide timely response to requests for code and configuration changes o Execute off-hours changes as necessary for emergency or sensitive deployments o Work with development staff to help them understand the deployment processes o Drive to a standardized process across all Post Consumer Brands applications o Facilitate the regular code review meeting o Maintain up-to-date code deploy process documentation o Cross-train on key code and configuration change activities Perform user account setups and removals for controlled systems in accordance with applications o Execute off-hours changes as necessary for involuntary terminations o Maintain up-to-date system access process documentation o Cross-train on key user account setup and removal
