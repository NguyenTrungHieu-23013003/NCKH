Position Responsibilities\Requirements: JOB SUMMARY : Responsible for the all Hosted Customer migrations as part of the Hosted Services Build Team, deploying the release of customer technologies as well as the design, installation, configuration, maintenance and system integration testing performance of UCS server & Netapp storage operating systems, related utilities and hardware. Performs activities for the Hosted Services customer migrations to include device configurations, internal operating system upgrades, and software upgrades; ensuring the newly built customer environment is transitioned into Hosted production. Works closely with the Hosted Solution Architects to analyze, design and implement customer base infrastructure. GENERAL DUTIES & RESPONSIBILITIES: Installs, maintains and upgrades internal computer hardware and software systems used for designing and developing company products. Researches, evaluates and recommends software and hardware products. Supports Web access and electronic messaging services, and maintains a secure systems environment. Provides new hardware specifications to users based on application needs and anticipated growth. Performs other related duties as assigned.
