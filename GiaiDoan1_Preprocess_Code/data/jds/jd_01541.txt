Overview The Data Migration Specialist conducts migrations of sensitive data from legacy systems while ensuring that data quality, integrity and completeness are maintained or improved. Responsibilities Collaborate with project managers and engineers to ensure high-quality data migrations are delivered on-time and provide post-implementation support Work with engineering teams to better understand systems and tools available to perform data migration. Audit and analyze data from third party vendor for accuracy, integrity, and ensure SLA adherance of 98.5% accuracy. Intermediate understanding of SQL statements & Database navigation per Ad-hoc requests Responsible for the profiling, cleansing, converting and loading of data for all projects assigned. Follow standard data validation processes to ensure data integrity and configured data meets compliance standards. Assist with the collaboration of solutions that will streamline the data migration processes Ensure legal, compliance, and IT Security policies are followed while maintaining complete confidentiality of PII. Communicate with Project Managers for any issues or exceptions during the data migration or integration process. Maintain up-to-date records in project management software Support business needs with ad hoc analysis or reports Qualifications Bachelor's Degree Clear, fundamental understanding of relational databases and industry-recognized concepts pertaining to SQL Server and/or Oracle database management systems Demonstrated technical experience with data and/or file analysis Experience writing and reviewing SQL (Structured Query Language) Experience developing solutions using core data migration technologies Development experience with at least one core technology used by Data Migration Able to read and write software documentation from a configuration or user interaction perspective Good collaborative skills, applied successfully within team as well as with other areas Able to thrive in a fast paced, deadline driven environment Self-motivated with the ability to manage projects to completion Speak or write with ease, clarity and impact, using a communication style appropriate to the subject and the audience Good analytical and problem solving skills Travel up to 10%
