This position performs the tasks necessary to analyze, design, configure, implement and support Costco’s voice network infrastructure, with primary focus on Cisco UC solutions. The position will require working closely with the Infrastructure and Application Development areas to develop detailed design and execution of strategic solutions. The position entails network monitoring and troubleshooting, cooperative engagement with other departments and vendors, working to resolve network, application, and configuration issues/concerns, and analysis and implementation of system changes.
