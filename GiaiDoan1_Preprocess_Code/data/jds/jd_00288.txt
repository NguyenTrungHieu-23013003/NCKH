Job overview We are looking for a Data Scientist with experience in Control Systems and a strong background in Computer Science. This person will be working with domain experts to collect and analyze our equipment’s data, and create statistical models and algorithms to optimize the operation of our machines. We are looking for a self-motivated and disciplined individual, who is open and enjoys working with other people. Specific duties to include: Use existing software and write new code to collect and store Process and Machine Data - work with other Software engineers to use 3rd party tools and design new ones to collect and persist large volumes of machine data in a way that maintains the equipment’s performance Build statistical models and design algorithms for discovery and to improve machine’s performance and output, and reduce errors Work with software architects to create frameworks and methods for real time data analysis and real-time decision. Requirements Proven experience in data analysis of control systems (2+ years) Strong Computer Science background and proven programming experience (2+ years) Theoretic knowledge in Data analytics, Statistics and Machine Learning Experience with Data Analytics libraries (Python or R) Experience with Data Visualization libraries (High charts, D3, Bokeh) Some experience with web programming - javascript \ node.js Education Bachelor’s degree (or higher) in Computer Science or related degree, MS (or higher) preferred.
