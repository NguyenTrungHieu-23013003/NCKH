The System Administrator (SA) is responsible for effective provisioning, installation/configuration, operation, and maintenance of systems hardware and software and related infrastructure. This individual participates in technical research and development to enable continuing innovation within the infrastructure. This individual ensures that system hardware, operating systems, software systems, and related procedures adhere to organizational security policy standards. This is a full-time, permanent position with competitive salary and benefits. Must have a two year degree or equivalent/higher and have 4+ years experience with the following applications and processes: VMware Microsoft Windows Server OS 2016 General knowledge of various manufacturer server and storage hardware platforms (Nutanix, HPE, NetApp, CISCO, Dell, etc) Data Center and Local Area Networking. Microsoft Windows ActiveDirectory and Group Policy Management Ability to adapt to existing business applications Storage allocation and File Share Management Perform daily system monitoring, verifying the integrity and availability of all hardware, server resources, systems and key processes, reviewing system and application logs, and verifying completion of scheduled jobs such as backups. Host and Virtual server patching and vulnerability remediation Repair and recover from hardware or software failures. Work with DR/BCP committees to coordinate and communicate with impacted application owners. Ability to educate and mentor coworkers as needed Provide Tier III/other support per request from application owners. Investigate and troubleshoot issues.
