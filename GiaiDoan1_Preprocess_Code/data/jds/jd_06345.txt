The Systems Integration Engineer II is a technical position in the IT Applications Support group. The primary responsibilities include the coding of standard and non-standard HL7 interfaces and file transfers through an interface engine product Orion Rhapsody. Working with IT Application Analysts and external software vendors, the Systems Integration Engineer II determines the document interface requirements, and builds and configures data translators, connectors and routers. Responsibilities include the deployment, testing and maintenance of real time and batch interfaces within the Orion Rhapsody environment as well as installation, upgrade and configuration of the Orion Rhapsody engine itself. In addition to system integration, this position may involve the installation and support of other application software, as well as the configuration and maintenance of Microsoft SQL databases. QUALIFICATIONS: Bachelor's degree with substantial coursework in Information Technology, Computer Science, Business Administration or Management, or equivalent education and training. Minimum 2 related work experience. Skills and demonstrated knowledge in HL7, Java or JavaScript, NT Scripts, Microsoft SQL, TCP/IP, Windows Server, Visio, Visual Basic, Microsoft Office, HTML, XML. Experience in a healthcare setting, especially regarding the flow of clinical data within acute care hospital environment. Must demonstrate excellent organizational skills and ability to work independently. Must demonstrate strong verbal and written English communication skills.
