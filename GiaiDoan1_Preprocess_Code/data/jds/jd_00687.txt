The analyst position will be pivotal in analyzing and reporting of GLOBO's key performance metrics and will aid in ongoing optimization efforts. The position will provide the opportunity to work closely with all the departments of GLOBO. The Data Analyst will receive on the job training to tackle problems that require knowledge of SQL and Python Responsibilities: Provide on-going analytical support utilizing data management, technology and modeling Analyze data and generate graphics and insights for use in client presentations. Assist in and formulate econometric models using regression analysis. Act as primary day-to-day contact on customer insights Manage project timelines and deliverable schedules and share concerns about deliverables, timelines, and issues with project managers and directors. Requirements: Currently enrolled in a technical field, such as Computer Science, Mathematics, Statistics, Economics, Management Information Systems, Physics, Business Analytics or related fields Strong analytical skills; ability to manage, manipulate and analyze raw data, draw conclusions, and develop actionable recommendations Experience in writing SQL queries is a big plus Ability to work in a fast-paced environment focusing on real time analytics. Knowledge of visual techniques for data analysis and presentation. Strong interpersonal written and verbal communication skills. Strong skills in Excel. Bonus Points: Understanding of SAS, R or any other advanced analytics software packages Experience using Looker or similar BI tool Basic proficiency in programming languages such as Python, Ruby or Java Double points if you use the Python library scikit-learn http://scikit-learn.org/stable/index.html [http://scikit-learn.org/stable/index.html] or similar
