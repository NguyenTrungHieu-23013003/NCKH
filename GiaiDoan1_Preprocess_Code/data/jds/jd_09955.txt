Responsibilities Review network security service requests and provide recommendations Manage/Coordinate network security service requests to ensure appropriate resources are engaged Create and maintain evergreen processes for network security service requests processing Create and maintain analytics and reporting Research and stay up to date on emerging technologies and techniques to improve network security program. Support remediating network security control deficiencies in a timely & pragmatic way Help grow and mature the network perimeter security program and technologies within Enterprise Information Security Maintaining familiarity with industry trends and security best practices Build close relationships with peers within Enterprise Information Security Closely collaborate with security architects in developing network security architectural principles, policies, reference stacks, features, and requirements.
