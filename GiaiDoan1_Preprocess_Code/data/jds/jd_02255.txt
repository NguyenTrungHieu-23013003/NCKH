Job Description The SQL Server Database Administrator position manages multiple databases for the airport’s development and production environments. This position is responsible for database creation, organization, installation, data migration, maintenance, support, backup and restoration of the database environments and its applications. The SQL Server Database Administrator will work with the IT and user groups to maintain and support the database environments. Essential Functions Communicates with managerial, administrative and technical staff to determine information needs, data flows and systems definitions Manages the environments, distribution and retention of data to ensure data integrity Manages and administrates the data structures and maintains an inventory of data definitions Develops policies and procedures related to the support and maintenance of the database and applications Performs database turning to maintain or increase database performance Assist and advise in the performance of tuning SQL queries Provides input for strategic planning, high availability and disaster recovery Performs database administration duties, including installation, configuration patching, recovery management, space management, schema management, startup and shutdown. Ensures all databases are backed up to meet Airport’s recovery objectives Test backups to ensure Airport’s recovery objectives can be met Reviews proposed changes to database to evaluate effects on logical and physical structure and space, security and data integrity Monitoring SQL Server to minimize outages and troubleshooting to restore outages as they occur Write database change scripts and deploy database change scripts including those from manufacturers and vendors Other duties as assigned. Competencies, Knowledge, Skills and Abilities Considerable working knowledge of and experience with SQL server database environment including: o Microsoft SQL Server 2012, 2016 Enterprise and Azure o SQL Server Management Studio o SSRS, SSIS and Report Builder o Data Extract, Transform and Load tools o T-SQL Considerable working knowledge of operating system environments including: o Windows 2008/20012/2016 o Linux OS – SUSE, RedHat o Shell scripting Working knowledge or ability to learn the Oracle database environment Working knowledge of Windows 2008, 2012 and 2016 Server operating environment Maintenance of Database integrity and various interfaces Enforcement of Quality Control and Database standards in the deployment and development of applications Enforcement of information security policies Management of the various SLAs (Service Level Agreements) Knowledge of SQL recovery tools and utilities and SQL enterprise management tools and utilities Proficiency in T-SQL and/or PL/SQL Knowledge of JavaScript, JAVA, J2EE and XML
