# 🎓 Scientific Research Upgrade: CV-JD Hybrid Matching (NCKH)

This document formalizes the methodology and technical improvements implemented to meet international research standards for AI-based Recruitment systems.

## 1. Mathematical Formalization of the Hybrid Model

To improve scientific clarity in the paper, the matching score is defined as follows:

$$Score_{Hybrid}(JD, CV) = \alpha \cdot \text{Sim}_{Semantic}(JD, CV) + (1 - \alpha) \cdot \text{Sim}_{Keyword}(JD, CV) - \text{Penalty}_{Category}$$

### Component Definitions:
1.  **$\text{Sim}_{Semantic}$**: Cosine similarity measured in the latent embedding space of a fine-tuned **Multilingual-E5-small** model (transformed via SBERT architecture).
2.  **$\text{Sim}_{Keyword}$**: Cosine similarity of global **TF-IDF vectors**. 
    - *Crucial Optimization*: The TF-IDF Vectorizer is fitted once on the entire dataset (Jobs + Resumes) to ensure the Inverse Document Frequency (IDF) correctly identifies domain-specific keyword rarity (e.g., "Python" vs. "Teamwork").
3.  **$\alpha$ (Weighting Factor)**: Empirically set to **0.7**, prioritizing semantic context while retaining keyword verification.
4.  **$\text{Penalty}_{Category}$**: A heuristic penalty applied if the predicted industry category of the CV (CV Parser) does not match the JD category.
    - $Penalty = 0$ if $Category_{JD} = Category_{CV}$.
    - $Penalty = 0.4 \times Score_{Total}$ if $Category_{JD} \neq Category_{CV}$.

## 2. Statistical Validation Framework

### A. Pearson & Spearman Correlation
We evaluate the model using Ground Truth (Expert Scores) provided by HR Professionals.

| Metric | Business Meaning |
| :--- | :--- |
| **Pearson Correlation ($r$)** | Measures how linearly the AI scores follow human scores. Higher $r$ means the AI mimics human scoring scales. |
| **Spearman Correlation ($\rho$)** | Measures ranking alignment. High $\rho$ ensures top-ranked candidates by AI are indeed the best according to humans. |

### B. Inter-Annotator Agreement (IAA)
When using multiple HR experts (e.g., Expert A and Expert B), researchers should calculate **Cohen’s Kappa ($\kappa$)** or **Fliess' Kappa** to measure consistency.

**Formula for Cohen's Kappa:**
$$\kappa = \frac{p_o - p_e}{1 - p_e}$$
- $p_o$: Observed agreement.
- $p_e$: Hypothetical probability of chance agreement.
